De Baker's & More Website

As a website designer, I had the privilege of creating a visually appealing and user-friendly website for a De Baker's & More, and it was an exciting and fulfilling project. The bakery's website aimed to capture the essence of its delectable offerings and the warmth of its ambiance.The website features a clean and inviting design with a color palette that reflects the bakery's brand identity—soft pastels and warm earthy tones that evoke a sense of comfort and indulgence. The homepage greets visitors with high-resolution images of freshly baked treats, enticing them to explore further.

First React Project

TextUtils - Text Transformer

TextUtils is an text transform web application created with the help of ReactJS which uses the the input text to transform it into an Uppercase, Lowercase, Removes Extra Spaces from the given text paragraph or line and also uses text to speach translation. This was the very first project made by me on ReactJS. This project was built along with CodeWithHarry's guidance.
